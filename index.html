Bienvenido al curso de Matemáticas

Este sitio web es un curso gratuito y completo que reúne todo el contenido de una carrera de matemáticas convencional, basado en los planes de estudios impartidos en España. Nuestra misión es proporcionar acceso abierto a una formación matemática de calidad para estudiantes, autodidactas y entusiastas.

Aquí encontrarás contenido detallado y estructurado por cursos, incluyendo apuntes, demostraciones, ejemplos y ejercicios. El objetivo es que tengas a tu disposición todos los recursos necesarios para aprender matemáticas de forma rigurosa y organizada.

Explora los cursos

Usa el menú para navegar entre los cursos disponibles. Cada sección está diseñada para cubrir las materias específicas de cada año académico.
